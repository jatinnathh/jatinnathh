### TurboS / Mediflow — Hospital Workflow Automation System

Full-stack healthcare system managing inter-department workflows in real time.

* Next.js + TypeScript + Prisma + PostgreSQL (NeonDB)
* Role-based system: Patients, Doctors, Staff, Admin
* Real-time communication via Socket.IO
* End-to-end lifecycle:
  patient → doctor → lab → prescription → discharge

**Core Features:**

* Automated doctor assignment based on availability
* Cross-department referrals with full audit trail
* Lab test pipeline with status tracking
* Prescription and billing integration
* Real-time activity logs + workflow visualization
* AI-assisted diagnosis (Gemini + ML models)

🔗 https://turbo-s-final.vercel.app/
